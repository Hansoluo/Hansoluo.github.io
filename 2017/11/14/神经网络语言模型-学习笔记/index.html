深度学习笔记1-神经网络语言模型

Nov 14, 2017

理解了词向量，才算真正进入NLP世界。

NLP的基础概念就是词向量了，CNN、RNN、LSTM等复杂模型都以此为基础。NLP发展到现在，基本已经确定了以统计为核心，而摒弃了以语法为中心。统计属于数学的一个学科。若要以数学来处理，少不得要对对象进行数学建模。词向量即用向量去表示一个词汇，而方便复杂数学模型的处理。

重点

  • 理解神经网络的矩阵表示形式
  • 了解神经网络语言模型的优势
  • 理解 Word embedding 的概念（中文可解释为 “词嵌入”，Word embedding 作用是将源数据映射到另一个空间）

视频1：更深入的理解神经网络

  • 深度学习算法本质还是梯度下降
  • 以一个最简单的神经网络来推导w和b
  • 链式法则 -> 剥洋葱 -> 反向传播算法
  • 矩阵乘法

视频2：神经网络的表达能力

  • 表达能力指的能够表达什么样的x到y的映射关系（函数）
  • 理论可证明单隐层有限的节点数可以逼近任意形状的函数，但实际应用不可取，节点数庞大
  • 隐层越多，结果跟准确，运算更快
  • 以如何学画画举例（类似于认知学中的组块理论）

视频3：矩阵符合表示神经网络

  • W * X -> 输入数据X表达为列向量
  • 多样本的矩阵乘法
  • X * W -> 输入数据 X 表达为行向量
  • tf.layers.dense 类似于 矩阵乘法

视频4：神经网络语言模型

  • 统计语言模型存在的问题
    • N-grama 无法判断两个很像的词 如今天 和 明天
    • 本质是只靠查字典方式，无法推理和迁移
  • 神经网络语言模型
    • 用于情感分类的二分类问题是输出一个概率，表示正面情绪的强度即可
    • 语言模型要素：
      • 输出一个概率分布，有N个词，就有N个输出
      • 加和等于1
    • sigmoid -> softmax 归一化指数函数
      • 对数据进行归一化处理，所有数据和为1
      • 权重保持不变，凸显大的值
    • 输入：上下文
    • 词向量(word embedding)->权重的向量
    • 稀疏矩阵
  • 为什么NN_LM可以迁移知识？
    • 所有样本进行全局优化
    • 今天和明天在相似的上下文里面，词向量类似
    • 今天有而明天没有的一些上下文，明天可以GET到
