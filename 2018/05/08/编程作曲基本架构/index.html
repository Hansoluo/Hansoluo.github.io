JythonMusic学习笔记

May 8, 2018

这篇文章是关于JythonMusic网站的学习笔记。

Overview页面介绍

JythonMusic可以看作是编曲API，它可以实现像程序员编写代码一样来完成音乐的作曲。开发者认为编曲和编程在本质是相同的。

JythonMusic提供了作曲的音乐库，并提供GUI界面和支持连接外部设备如电子琴、智能手机和平板等等。

JythonMusic基于python语言，适合各类音乐爱好者，无论你是大师还是新手。

JythonMusic被用于音乐作品的教学、创作和研究。

Download页面介绍

  • 给出了下载链接，对于windows用户需要提前安装Java运行环境。
  • 解压后，双击JEM editor，类似于代码编辑器
    • 主界面是文本编辑的界面，可以用来写和修改代码
    • 工具栏有运行代码按钮
    • 底下是编译代码的界面
  • 可以使用命令行

提供了很多代码示例

Reference页面

  • 这部分是API文档页面，提供了各类函数库的调用和接口说明
  • Music Library
    • MIDI Constants：将音乐中的基本元素转化成数字
      • pitch 音高
      • duration 音程
    • Transcription 编曲
      • Note 音符
      • Phrase 短章
      • Part 段落
      • Score 完整的曲子
    • Composition 组合
      • Mod 对乐曲进行各种变换
      • View 乐曲的可视化
      • Read / Write 读写MIDI文件
    • Performance
      • Play 播放
      • AudioSample 处理音频材料
      • LiveSample 处理现场音频材料
      • MidiSequence 播放外部MIDI文件
      • Metronome 节拍器
    • Mapping Values 映射值
