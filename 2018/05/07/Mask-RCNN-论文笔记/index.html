Mask R-CNN 论文笔记

发表于 2018-05-07
字数统计: 687字

Abstract

  • 在 Faster R-CNN 的预测目标边界框的基础上增加一条并行预测目标物体掩膜的分支
  • 关键词： instance segmentation / Faster R-CNN / person keypoint detction

Introduction

  • 计算机视觉领域要解决的问题
    • 目标检测的目标是对图像中物体进行识别并且定位边界框
    • 语义切割的目标是将图像中每一个像素归类到一个固定的物体类别中
    • 实例切割是两者的结合
  • Mask R-CNN 框架
    • Faster R-CNN
    • 对提取的RoI应用FCN来聚类像素
  • 难点
    • Faster R-CNN 将实际的像素图片转换成RPN的卷积特征图
    • 但RPN的卷积特征图和实际像素图片仍然差很多
    • 因此在整个框架的输入输出中如法实现pixel-to-pixel对准
    • 提出RoiAlign来实现空间上的对准

Related Work

  • Faster R-CNN
    • R-CNN 在特征图上直接提取RoI,提升速度和精度
    • Faster R-CNN 使用 RPN 进一步提升速度和精度
  • 实例切割
    • DeepMask 提出切割候选区域，再使用Faster R-CNN来分类，速度慢且精度差
    • FCIS 使用全卷积同时对目标类别、区域框和掩膜进行预测，速度快但精度差
      • 重叠区域多
      • 分割线不正确

Mask R-CNN

  • Faster R-CNN 分为两级训练
    • 第一阶段称为RPN，提出目标的候选框
    • 第二阶段是精简版的Fast R-CNN，使用RoIPool从第一阶段输入的候选框中提取特征进行分类和边界框回归
    • 两个阶段提取的特征可以实现共享，用来加快速度
  • Mask R-CNN
    • 思路
      • 两个阶段同Faster R-CNN，而且对提取出的每一个RoI输出二分的掩膜
      • 借鉴Fast R-CNN的思路，并行的去实现边界框的分类和位置回归（大大R-CNN的增强速度）
    • 损失函数定义
      • 三部分：分类+边界框+掩膜
      • 掩膜分支的输出为： Kmm 维度数量 (K 为类别的个数，m*m为分辨率)
      • 对每一个像素进行加上sigmoid激励输出
      • 再对总的交叉熵损失函数取平均为最后的损失函数
    • RoIAlign
      • RoIPool是一种从ROI中提取特征图的标准操作，其原理是将区域离散化到格子中，但是比较粗糙
      • 在RoIPool的基础上，使用双线性插值（bilinear interpolation）来实现对准
    • 网络架构
      • 原始的Faster R-CNN的主干网络使用 ResNet-50-C4
      • 本文使用 ResNet-FPN 来替代，可以应对各种尺寸的图片
      • 其他部分做了轻微的调整
