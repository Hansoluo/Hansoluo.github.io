<利用姿态归一深度卷积网络实现鸟分类> 论文笔记

May 13, 2018

CVPR 2014 Goggle 分享的目标检测综述中特别提及的论文：Bird Species Categorization Using Pose Normalized Deep Convolutional Nets，对处理图片分类问题处理有帮助。

Abstract

  • 先获得物体的姿态，用来生成姿态的图片
  • 再用DCNN来对姿态图片进行特征提取
  • 最好的模型整合了带有姿态归一提取的特征图和没有对准的高层级的图片
  • 关键词：pose normalization /fine-tuning feature / bird species recognition

Introduction

  • 流程描述（如下图）
    • 利用关键点群检测得到多种区域
    • 每个区域输入到对应的DCNN模型中
    • 对上述提取的特征进行连接，并输出给分类器
  • 本文贡献
    • 新的高阶图片裁剪和基于图的聚类算法用来进行姿态归一化
    • CNN模型训练的研究

Related Work

  • ImageNet数据集上预训练的CNN模型用于其他数据有很好的效果
  • CNN的不同层对应不同的姿态特征
  • CNN权重的fine tune

Pose Normalization Schemes

典型区域的姿态归一化

  • 用2D的关键点集作为图片关键区域的标记，
  • 使用三个方程对关键点集进行变换，来获取更多的特征
    • simple translations
    • similarity transformations (2D rotation, scale, and
      translation)
    • affine transformations

学习典型姿态

  • 目标是为了减少关键点和目标区域的对准错误
  • 用一个贪婪算法来实现最优化

Deep Convolutional Features

多重CNN网络用于实现不同的对准姿态

  • 8层CNN网络用来学习从低到高层次的特征
  • CNN网络层数越到后面越更加获取高级的结构型特征
  • 全连接层可以捕获复杂的共同的特征

对比fine-tune手法

  • 直接使用预训练模型
  • Fine-Tuning the ImageNet Model
    • 去掉分类层,1000分类换成200的分类输出层
    • 新层的权重数据是随机初始化的，因此是很难保持和之前预训练模型的收敛性，因此要用很低的学习速率，一般降低10倍
  • Two Step Fine-Tuning Method
    • 避免不同层使用不一致的学习速率
    • 冻结预训练模型，只训练新层（相当于使用预训练模型做特征提取，再进行多分类问题，速度快容易收敛，避免新层权重初始化太随机）
    • 训练所有层，保持相同学习速率
  • 从头训练
    • 从头训练相同的CNN模型，不使用Imagenet权重参数初始化，进行随机处理

Experiments

  • CUB-200-2011数据集：200鸟的种类、11,788张图片、15种关键点数据
  • 用DPM预测位置和区域的关键点
  • 基础学习速率 0.001

Conclusion

CNN

  • CNN方法比传统的HOG、SIFT方法提升很大
  • CNN的不同层用来不去区域的怼准

姿态归一

  • 相似度的算法最好
  • 多区域能提高成绩

fine tune手法对比

  • 微调CNN权重值可以提高成绩
    • 比较手法1和手法2，可以发现手法2的模型表现提高2~10%
  • 预训练模型很重要
    • 对比手法2和手法1，发现不用ImageNet模型的权重，模型表现很差
  • 方法3的模型表现最好
    • 方法3比方法2的表现要好2%
